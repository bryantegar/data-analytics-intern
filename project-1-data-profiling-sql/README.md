Project ini bertujuan melakukan data mining dengan scraping data harga komoditas dari sumber eksternal menggunakan Python dan BeautifulSoup. Data dibersihkan, diproses, dan disimpan menjadi dataset siap analisis.

Tools:
- Python
- BeautifulSoup
- Pandas

Outcome:
Dataset bersih yang dapat digunakan untuk analisis tren harga dan forecasting.
